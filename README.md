# CHOKQU WEB FRAMEWORK
Chokqu is a web framework for building web applications in Rust.

## Features
- Fast and lightweight
- Easy to use
- Built-in support for routing, middleware, and templating
- Asynchronous support
- Routing
- Middleware
- JSON support
